Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
header header make a heading
div div defines a division or a section in an HTML document to be either styled with CDD or manipulated with JavaScript.
ol Ordered List used to set off an ordered (i.e. numbered) list.
ul Unordered List used to set off an unordered (i.e. bulleted) list. Other uses possible with styling and css.
blockquote blockquote creates an indented section, quoted from another source
header header section represents introductory content, typically a group of introductory or navigational aids

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
span span an inline element used to style or color a part of a text <span [insert style info]> insert sample text </span> We have red apples.
li list item an inline element used to define an item in an ordered or unordered list <li>list item</li>>

Document Tags

Tag Name Purpose
html html Tells the browser that we are using HTML
head head Specifies the title and other information about the page
body document body Begins the body of the page
title title specifies the page title (viewed in the browser tab)
meta meta tag Specifies metadata information about the document, such as the character set
